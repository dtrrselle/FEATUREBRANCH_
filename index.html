Google Search

A simple web page demonstrating GitHub Feature Branch Workflow, Pull Requests, and GitHub Pages deployment.

Project Information

This website was developed as part of a collaborative GitHub activity. Each team member worked on a separate feature branch before merging their changes into the main branch.

Features

  • Simple and clean search interface
  • Redirects search queries straight to Google
  • Responsive layout for desktop and mobile
  • Built and deployed using GitHub Pages

How It Works

  1. Type your search term in the input box above.
  2. Click the "Google Search" button.
  3. You will be redirected to Google's search results page.

Workflow Steps

Each contributor created their own feature branch, made changes, and opened a Pull Request for review before merging into main. This ensured a clean commit history and avoided direct conflicts on the main branch.
